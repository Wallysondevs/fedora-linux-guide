# Guia Fedora Linux

Projeto web educacional com trilha completa para estudo de Fedora Linux.

## Requisitos

- Node.js 18+ (recomendado: versão LTS)
- npm

## Desenvolvimento local

```sh
npm install
npm run dev
```

## Build de produção

```sh
npm run build
npm run preview
```

## Testes

```sh
npm run test
```

## Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
